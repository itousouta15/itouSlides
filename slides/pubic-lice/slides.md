# 什麼是陰蝨？

---

## 陰蝨（Pubic Lice / Crabs）

陰蝨是一種**寄生在人體毛髮上的昆蟲**，屬於性傳播感染（STI）的一種

<br>

| 項目 | 說明                       |
| ---- | -------------------------- |
| 學名 | _Pthirus pubis_            |
| 俗名 | Crabs（螃蟹）              |
| 分類 | 節肢動物 — 蝨目            |
| 體長 | 約 1–2 mm                  |
| 宿主 | 僅人類，無法在其他動物存活 |

<br>

> 英文俗稱「Crabs」是因為它的外形在顯微鏡下長得像螃蟹

# 病原體介紹

---

## _Pthirus pubis_（蟹形陰蝨）

**外觀特徵：**

- 體長約 1–2 mm，肉眼勉強可見
- 外形扁平，前腳細小、中後腳粗壯有爪
- 顏色為灰白至棕色，吸血後變紅棕色

<br>

**生命週期：**

```
卵（蟲卵/nits）→ 若蟲（nymph）→ 成蟲
  附著毛根       約 6–10 天       約 2–3 週壽命
```

<br>

**離開宿主後最多存活 24–48 小時**，高度依賴人體溫度與血液維生

# 寄生部位

---

## 陰蝨住在哪裡？

**主要寄生部位（粗毛區域）：**

- 陰毛（最常見）
- 肛門周圍毛髮
- 腹部與大腿內側的體毛

<br>

**其他可能部位：**

- 腋毛
- 胸毛、腹毛
- 鬍鬚
- 眉毛、睫毛（少見，多見於兒童）

<br>

> **不**寄生於頭髮——頭髮的毛髮間距太密，陰蝨的爪子無法抓附

# 傳播途徑

---

## 怎麼感染到陰蝨？

**主要途徑：**

```
性接觸（皮膚對皮膚密切接觸）
  約佔 90% 以上的感染案例
```

<br>

**非性接觸途徑：**

| 途徑     | 說明                             |
| -------- | -------------------------------- |
| 共用寢具 | 床單、枕頭、棉被                 |
| 共用衣物 | 內褲、泳衣、毛巾                 |
| 肌膚接觸 | 非性行為的密切身體接觸（較少見） |

<br>

> 常見迷思：**不會經由馬桶座傳染**——陰蝨無法在光滑表面爬行，且離體後迅速死亡

---

## 傳播的難易度

**容易傳播的情境：**

- 與感染者有性接觸
- 共用未清洗的寢具或衣物

<br>

**不容易傳播的情境：**

- 短暫接觸（握手、擁抱）
- 坐在公共廁所馬桶上
- 游泳池（氯會快速殺死陰蝨）

<br>

> 性伴侶若未同步治療，幾乎必定會再度感染（乒乓效應）

# 症狀

---

## 感染陰蝨的症狀

**最主要症狀：強烈搔癢**

<br>

| 症狀           | 說明                                           |
| -------------- | ---------------------------------------------- |
| 搔癢           | 通常在感染後 1–3 週出現，夜間特別嚴重          |
| 藍灰色小斑點   | 蝨子叮咬後的皮下出血（maculae ceruleae）       |
| 毛根白點       | 蟲卵（nits）附著在毛根，外觀像頭皮屑但不易去除 |
| 肉眼可見的蟲體 | 仔細觀察可在毛根處看到緩慢移動的成蟲           |

<br>

**重要：約 30% 的感染者完全沒有症狀，卻仍具有傳染性**

---

## 搔癢為什麼在夜間更嚴重？

陰蝨是**夜行性**生物

<br>

```
白天：蟲體相對靜止，附著在毛根不動
夜間：蟲體開始活躍，頻繁叮咬吸血
  → 唾液中的物質引發過敏反應 → 搔癢加劇
```

<br>

> 因此很多感染者描述：「晚上完全無法入睡」

# 診斷方式

---

## 如何確診？

陰蝨的診斷相對簡單，**不需要血液或尿液檢查**

<br>

**診斷方法：**

| 方法                 | 說明                             |
| -------------------- | -------------------------------- |
| 肉眼觀察             | 直接在毛根附近尋找成蟲或蟲卵     |
| 放大鏡觀察           | 更清楚確認蟲卵是否附著在毛幹上   |
| 皮膚鏡（Dermoscopy） | 醫療機構使用，可精確辨認蟲體和卵 |

<br>

**注意：** 蟲卵（nits）牢固附著在毛幹，不像頭皮屑可以輕易撥落

# 治療方法

---

## 如何治療陰蝨？

**外用殺蟲藥（首選）：**

| 藥物                  | 使用方式                     |
| --------------------- | ---------------------------- |
| 百滅靈 Permethrin 1%  | 塗抹患處，靜置 10 分鐘後沖洗 |
| 馬拉松 Malathion 0.5% | 塗抹患處，靜置 12 小時後沖洗 |

<br>

**特殊部位：**

- **眼睫毛感染**：厚塗凡士林於睫毛，阻斷蝨子呼吸，持續 10 天
- 眼部不可使用殺蟲藥

---

## 治療注意事項

**重要原則：**

1. **性伴侶需同時治療**（即使沒有症狀）
2. 治療後約 1 週回診確認是否清除乾淨
3. 若治療失敗，考慮換用不同藥物

<br>

**環境處理：**

```
寢具、衣物、毛巾 → 60°C 以上熱水清洗並烘乾
無法清洗的物品 → 密封於塑膠袋靜置 72 小時
```

<br>

> 蟲卵比成蟲更難殺滅，第一次治療後通常需要 7–10 天再治療一次

# 預防方式

---

## 如何預防陰蝨？

**個人預防：**

- 避免與感染者的密切皮膚接觸
- 不共用貼身衣物、毛巾、寢具
- 性行為使用保險套（**注意：無法完全防止**，因皮膚接觸仍可傳播）

<br>

**定期檢查：**

- 若有多個性伴侶，建議定期進行 STI 篩檢
- 發現症狀（搔癢）應立即就醫，避免繼續傳播

<br>

> 陰蝨感染**不代表不衛生**，任何有性行為的人都可能感染

# 為什麼「冷門」？

---

## 感染率下降的原因

**近 20 年全球陰蝨感染率大幅下降**

<br>

**主要原因 — 陰毛剃除習慣普及：**

```
陰蝨需要附著在毛幹上產卵 → 沒有毛 → 無法寄生
```

多項研究顯示，陰毛剃除（或蠟除）的流行與陰蝨感染率下降有直接相關性

<br>

**其他原因：**

- 性教育普及，民眾對 STI 的意識提升
- 治療藥物容易取得
- 就醫門檻降低

---

## 為什麼容易被誤診？

**常見誤診情況：**

| 被誤診為       | 原因                     |
| -------------- | ------------------------ |
| 濕疹           | 搔癢與皮膚發紅的外觀相似 |
| 過敏性皮膚炎   | 症狀表現相近             |
| 毛囊炎         | 患處可能出現紅疹         |
| 焦慮引起的搔癢 | 夜間搔癢被認為是心理因素 |

<br>

**羞恥感導致延誤就醫：**

> 許多患者因為「不好意思」，平均延遲 **2–4 週**才就醫，期間持續傳播給他人

## 今天的重點回顧

- **陰蝨** = 寄生蟲（節肢動物），非細菌也非病毒
- **主要傳播**：性接觸皮膚對皮膚，也可經由寢具衣物
- **馬桶座不會傳染**——常見迷思
- **症狀**：強烈搔癢（夜間更嚴重）+ 藍灰色小斑點
- **診斷**：肉眼即可確認，不需血液檢查
- **治療**：外用百滅靈/馬拉松，**性伴侶須同時治療**
- **冷門原因**：陰毛剃除習慣普及 + 羞恥感導致延誤就醫

<br>

> 性健康教育的目的，是讓人能夠做出有知識基礎的判斷與選擇
