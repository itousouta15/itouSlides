## 陰蝨（Pubic Lice）

陰蝨是一種**寄生在人體毛髮上的昆蟲**，屬於性傳播感染（STI）

| 項目 | 說明            |
| ---- | --------------- |
| 學名 | _Pthirus pubis_ |
| 俗名 | Crabs（螃蟹）   |
| 分類 | 節肢動物 — 蝨目 |
| 體長 | 約 1–2 mm       |

> 英文俗稱「Crabs」因外形在顯微鏡下像螃蟹

## _Pthirus pubis_

- 體長 1–2 mm，肉眼勉強可見
- 外形扁平，中後腳粗壯有爪，可緊抓毛幹
- 顏色灰白至棕色，吸血後變紅棕色
- 離開宿主後最多存活 **24–48 小時**

**生命週期：**

卵（nits）→ 若蟲（6–10 天）→ 成蟲（壽命約 2–3 週）

# 寄生部位

---

## 陰蝨住在哪裡？

<div class="grid grid-cols-2 gap-8 mt-4">
<div>

**主要部位**

- 陰毛（最常見）
- 肛門周圍毛髮
- 腹部與大腿內側體毛

</div>
<div>

**其他可能部位**

- 腋毛、胸毛
- 鬍鬚
- 眉毛、睫毛（少見）

</div>
</div>

<br>

> **不**寄生於頭髮——頭髮間距太密，陰蝨爪子無法抓附

# 傳播途徑

---

## 傳播方式

| 類型     | 途徑                         |
| -------- | ---------------------------- |
| 性接觸   | 皮膚對皮膚密切接觸（約 90%） |
| 共用寢具 | 床單、枕頭、棉被             |
| 共用衣物 | 內褲、泳衣、毛巾             |

**常見迷思：**

| 迷思         | 事實                           |
| ------------ | ------------------------------ |
| 馬桶座會傳染 | 不會——離體後迅速死亡，無法爬行 |
| 游泳池會傳染 | 不會——氯會快速殺死陰蝨         |

---

## 傳播注意事項

> 性伴侶若未同步治療，幾乎必定**再度感染**（乒乓效應）

<br>

**容易傳播的情境：**

- 與感染者有性接觸
- 共用未清洗的寢具或衣物

**不容易傳播的情境：**

- 短暫接觸（握手、擁抱）
- 一般公共場所接觸

# 症狀

---

## 感染陰蝨的症狀

| 症狀         | 說明                               |
| ------------ | ---------------------------------- |
| 強烈搔癢     | 感染後 1–3 週出現，夜間特別嚴重    |
| 藍灰色小斑點 | 叮咬後皮下出血（maculae ceruleae） |
| 毛根白點     | 蟲卵（nits）附著毛根，不易去除     |
| 肉眼可見蟲體 | 毛根處可見緩慢移動的成蟲           |

<br>

> 約 **30%** 的感染者**完全無症狀**，卻仍具有傳染性

---

## 為什麼夜間搔癢更嚴重？

陰蝨是**夜行性**生物

<br>

- 白天：蟲體靜止，附著毛根不動
- 夜間：開始活躍，頻繁叮咬吸血
- 唾液中的物質引發過敏反應 → 搔癢加劇

<br>

> 很多感染者描述：「晚上完全無法入睡」

# 診斷方式

---
layout: image-left
image: /2.jpg
---

## 如何確診？

不需要血液或尿液檢查

| 方法     | 說明                       |
| -------- | -------------------------- |
| 肉眼觀察 | 在毛根附近尋找成蟲或蟲卵   |
| 放大鏡   | 確認蟲卵是否附著在毛幹     |
| 皮膚鏡   | 醫療機構使用，精確辨認蟲體 |

> 蟲卵牢固附著毛幹，**不像頭皮屑可輕易撥落**

# 治療方法

---

## 藥物治療

| 藥物                  | 使用方式                     |
| --------------------- | ---------------------------- |
| 百滅靈 Permethrin 1%  | 塗抹患處，靜置 10 分鐘後沖洗 |
| 馬拉松 Malathion 0.5% | 塗抹患處，靜置 12 小時後沖洗 |

**眼睫毛感染：** 厚塗**凡士林**阻斷蝨子呼吸（眼部不可用殺蟲藥）

---

## 治療注意事項

**重要原則：**

1. **性伴侶需同時治療**（即使無症狀）
2. 治療後 1 週回診確認是否清除乾淨
3. 治療失敗則換用不同藥物

**環境處理：**

- 寢具、衣物 → 60°C 以上熱水清洗並烘乾
- 無法清洗的物品 → 密封塑膠袋靜置 72 小時

> 蟲卵比成蟲難殺，通常需 **7–10 天後再治療一次**

# 預防方式

---

## 如何預防陰蝨？

- 不共用貼身衣物、毛巾、寢具
- 使用保險套（無法完全防止，皮膚接觸仍可傳播）
- 若有多個性伴侶，定期進行 STI 篩檢
- 發現症狀立即就醫，避免繼續傳播

<br>

> 陰蝨感染**不代表不衛生**——任何有性行為的人都可能感染

# 為什麼越來越少見？

---

## 感染率下降的原因

**近 20 年全球陰蝨感染率大幅下降**

主因：**陰毛剃除習慣普及**

陰蝨需附著毛幹產卵，沒有毛就無法寄生

其他原因：

- 性教育普及，民眾對 STI 意識提升
- 治療藥物容易取得

---

## 為什麼容易被誤診？

| 誤診為       | 原因                     |
| ------------ | ------------------------ |
| 濕疹         | 搔癢與皮膚發紅相似       |
| 過敏性皮膚炎 | 症狀表現相近             |
| 毛囊炎       | 患處出現紅疹             |
| 焦慮         | 夜間搔癢被認為是心理因素 |

> 許多患者因羞恥感，平均延遲 **2–4 週**才就醫，期間持續傳播

## 回顧

- **陰蝨** = 節肢動物寄生蟲，非細菌也非病毒
- **傳播**：性接觸為主，也可經由寢具衣物
- **馬桶座不傳染**——常見迷思
- **症狀**：強烈搔癢（夜間更嚴重）、藍灰色小斑點
- **診斷**：肉眼即可確認，不需血液檢查
- **治療**：外用百滅靈 / 馬拉松，**性伴侶須同時治療**
- **冷門原因**：陰毛剃除習慣普及、羞恥感延誤就醫

> 性健康教育讓人能做出有知識基礎的判斷與選擇
