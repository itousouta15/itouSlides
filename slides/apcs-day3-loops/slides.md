## 沒有迴圈的世界

# for 迴圈

---

## for 迴圈三元素

```cpp
for (int i = 1; i <= 5; i++) {
    cout << i << " ";   // 1 2 3 4 5
}
```

```
for ( 初始值 ; 條件 ; 更新 )
       i = 1   i <= 5   i++
```

- **初始值**：迴圈開始前先做一次（`int i = 1`）
- **條件**：每次執行前檢查，成立才繼續（`i <= 5`）
- **更新**：每跑完一輪做一次（`i++`）

---

## for 迴圈的執行順序

```cpp
for (int i = 1; i <= 3; i++) {
    cout << i;
}
```

| 步驟 | 動作       | i 的值 | 條件 i<=3 | 輸出 |
| ---- | ---------- | ------ | --------- | ---- |
| 1    | 初始化 i=1 | 1      | ✅ 成立   | 1    |
| 2    | i++ → 檢查 | 2      | ✅ 成立   | 2    |
| 3    | i++ → 檢查 | 3      | ✅ 成立   | 3    |
| 4    | i++ → 檢查 | 4      | ❌ 結束   | —    |

---

## 經典應用：累加求和

```cpp
// 計算 1 + 2 + 3 + ... + 100
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum += i;          // sum = sum + i
}
cout << sum;           // 5050
```

<br>

> 重點：先準備一個 `sum = 0` 在迴圈**外面**，迴圈裡面不斷累加。

# while 與 do-while

---

## while 迴圈

```cpp
// 先判斷，再執行
int n = 1;
while (n <= 5) {
    cout << n << " ";   // 1 2 3 4 5
    n++;                // 別忘了更新，否則無窮迴圈！
}
```

- 適合「不知道要跑幾次，只知道結束條件」的情況
- ⚠️ 一定要有讓條件「變 false」的那一步，否則永遠跑不完

---

## do-while 迴圈

```cpp
// 先執行，再判斷（至少執行一次）
int x = 10;
do {
    cout << "至少印一次" << endl;
    x++;
} while (x < 5);     // 條件一開始就 false，但仍印了一次
```

<br>

> 差別：`while` 先檢查條件，`do-while` 先做一次再檢查。

---

## 三種迴圈怎麼選？

| 迴圈       | 特性                     | 適用時機               |
| ---------- | ------------------------ | ---------------------- |
| `for`      | 次數明確                 | 跑固定次數、走訪陣列   |
| `while`    | 先判斷，可能一次都不執行 | 次數未知、看條件       |
| `do-while` | 先執行一次，再判斷       | 至少要做一次（如選單） |

<br>

> 三種其實可以互換，先把 `for` 練熟就夠用了。

# 巢狀迴圈

---

## 迴圈裡面再放迴圈

```cpp
for (int i = 1; i <= 3; i++) {       // 外層：列
    for (int j = 1; j <= 3; j++) {   // 內層：行
        cout << i << "," << j << "  ";
    }
    cout << endl;
}
```

```
1,1  1,2  1,3
2,1  2,2  2,3
3,1  3,2  3,3
```

> 外層跑一次，內層就整圈跑完一輪。

---

## 應用：九九乘法表

```cpp
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        cout << i << "*" << j << "=" << i * j << "\t";
    }
    cout << endl;
}
```

```
1*1=1   1*2=2   1*3=3   ...
2*1=2   2*2=4   2*3=6   ...
...
```

---

## 應用：印出三角形

```cpp
int n = 5;
for (int i = 1; i <= n; i++) {       // 第 i 列
    for (int j = 1; j <= i; j++) {   // 印 i 個星號
        cout << "*";
    }
    cout << endl;
}
```

```
*
**
***
****
*****
```

> 關鍵：內層的次數 `j <= i` 跟著外層的 `i` 變化。

# break 與 continue

---

## break：跳出迴圈

```cpp
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;     // i 等於 5 時，整個迴圈直接結束
    cout << i << " ";      // 1 2 3 4
}
```

## continue：跳過這一輪

```cpp
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;  // 偶數就跳過，不執行下面
    cout << i << " ";          // 1 3 5 7 9
}
```

# 實作練習

---

## 練習一：質數判斷

> 輸入一個大於 1 的整數，判斷是不是質數。

```cpp
int n;
cin >> n;
bool isPrime = true;

for (int i = 2; i < n; i++) {
    if (n % i == 0) {     // 能被 i 整除 → 不是質數
        isPrime = false;
        break;            // 找到一個因數就可以停了
    }
}

cout << (isPrime ? "是質數" : "不是質數");
```

---

## 練習二：累乘（階乘）

> 輸入 n，計算 n! = 1 × 2 × 3 × ... × n。

```cpp
int n;
cin >> n;

long long fact = 1;       // 階乘成長很快，用 long long
for (int i = 1; i <= n; i++) {
    fact *= i;
}
cout << fact;             // 例如 n=5 → 120
```

<br>

> 累乘的初始值要設 `1`（不是 0，否則全部變 0）。

---

## 練習三：數字翻轉

> 輸入 1234，輸出 4321。

```cpp
int n;
cin >> n;
int rev = 0;

while (n > 0) {
    int digit = n % 10;     // 取出個位數
    rev = rev * 10 + digit; // 接到結果後面
    n /= 10;                // 去掉個位數
}
cout << rev;
```

> `% 10` 取個位、`/ 10` 去個位，是處理「逐位數字」的萬用招。

## 今日重點回顧

- **for**：次數明確；**while**：看條件；**do-while**：至少一次
